\subsection{Detailed Description}
produces a lightweight vizualization for a graph. 

The quickrun utilty is intended for producing generic vizualisations for network graphs, relatively quickly using basic algorithms. \mbox{\hyperlink{struct_node}{Node}} sizes are produced using basic degree. \mbox{\hyperlink{struct_node}{Node}} (rgb) colours are delineated using walktrap modularity. \mbox{\hyperlink{struct_node}{Node}} positions are 2d and set using the Fruchterman-\/\+Rheingold algorithm, which emphasizes the spacing between nodes.
